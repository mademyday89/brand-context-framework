# Brand Context Framework

Digitales Assessment-Tool basierend auf Olaf Kopps 3-Phasen Methodologie.

## Features
- 12 Hauptfragen + 3 Expert-Fragen
- Animierter Login mit Glass-Card Design
- Radial Charts + Bar Chart Auswertung
- Responsive Design (Mobile/Desktop)

## Live Demo
- GitHub Pages: https://mademyday89.github.io/brand-context-framework/
- Passwort: `Social Marketing 2026`

## Technologie
- React 18 + Tailwind CSS
- Chart.js für Visualisierungen
- Glass-Morphism Design
